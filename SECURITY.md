# Security

Dieses Repo ist eine kuratierte öffentliche Projektunterlage.

Bitte keine Kundennamen, Rohtranskripte, Kontakte, internen Notizen, Prompts oder nicht anonymisierten Fahrpläne in Issues, PRs oder Diskussionen posten.

Wenn ein Sicherheits- oder Datenschutzproblem vermutet wird, nicht öffentlich diskutieren. Robert direkt kontaktieren.
